# remark-holding-test
       
编译：

1、编译选项说明：
   
   CMAKE_BUILD_TYPE：编译类型（默认Debug，可选Release）

2、编译命令

进入build文件夹执行如下命令

2.1 windows

cmake .. -LA -DCMAKE_BUILD_TYPE=Debug

2.2 linux

cmake .. -LA -DCMAKE_BUILD_TYPE=Debug

make
